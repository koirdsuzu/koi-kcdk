\# koi-KCDKについて



\* KCDK / Kojosen Command Development Kit / 攻城戦コマンド開発キットは、攻城戦（攻守に分かれて戦うゲーム）を簡単に作成する目的で作られたコマンドセットです。



\## 概要

\*\*koi.kcdk\*\* は、Minecraft における  

\*\*攻城戦（拠点攻略・防衛イベント）向けに設計されたコマンド群\*\* です。



プレイヤー制御・戦況管理などを  

コマンドブロックやデータパックと組み合わせて使用することを想定しています。

\## 主な用途

\- 攻城戦イベントの運営

\- チーム対抗の拠点戦

\- 防衛・侵攻フェーズの制御

\- 管理者による戦況リセット・調整



---



\## 特徴

\- 攻城戦に特化したコマンド設計

\- 使い回ししやすいシンプルな構造

\- コマンドブロック運用を前提

\- サーバー・シングルプレイ両対応（シングル...？）



---



\## 想定環境

\- Minecraft Java Edition

\- 対応バージョン: \*\*1.20.x 以降（想定）\*\*

\- 権限: OP または同等権限必須



※ バージョン差異によるコマンド挙動の違いにはご注意ください。



---



\## 構成内容

\- 攻城戦用コマンド群

\- 制御用スコアボード

\- 実行・リセット・管理向けコマンド群

---



\## 使い方（概要）

1\. インストール

2\. 攻城戦用のワールド・エリアを準備

3\. 戦闘開始時のtpとspawnpointの座標の設定

4\. 状況に応じて管理コマンドを使用

\## 注意事項
